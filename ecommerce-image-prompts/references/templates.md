# Ecommerce Prompt Templates

## Product Lock Prompt

```text
【Product Lock Prompt】
保持产品主体与参考图完全一致：产品的整体形状、尺寸比例、颜色、材质、边缘轮廓、正面结构、侧面结构、顶部结构、底部结构、按钮位置、接口位置、挂孔位置、开孔数量、纹理细节和品牌标识位置必须保持不变。不得改变产品类别，不得新增屏幕、按钮、接口、灯带、纹理、支架、线缆、配件或任何未在参考图中出现的结构。不得改变产品的核心轮廓、厚度比例、部件位置和材质表现。只允许调整拍摄角度、光影、场景、背景、构图和画面氛围，不允许臆造产品结构。
```

Customize this template with concrete product details observed in the user's product image. Mark unclear details as `需用户确认`.

## Style Lock Prompt

```text
【Style Lock Prompt】
整体视觉风格保持[风格气质]，画面[干净/真实/高级/生活方式/专业]，背景不抢产品主体。光线[方向和软硬]，产品边缘[是否有轮廓光]，突出产品材质和结构细节。色彩体系[克制/明亮/低饱和/高对比/冷暖关系]，避免杂乱和过度饱和。构图[简洁/留白/中心/对角线/英雄视角]，主体明确，画面具有电商广告图的专业感。参考图只借鉴构图、光影、色彩、镜头语言和视觉气质，不复刻具体产品、品牌元素、logo、包装、人物、道具组合或独特版式。
```

## Hero or Single Image Prompt

```text
【图片目标】
生成一张[图片类型]，用于[平台/用途]，突出[本张图核心卖点]。

【画面内容】
[描述产品、场景、背景、光影、氛围、主体关系。主视觉阶段以产品为绝对主体；后续图片根据卖点加入必要场景或道具。]

【产品锁定】
[粘贴或精炼复用 Product Lock Prompt。]

【风格锁定】
[粘贴或精炼复用 Style Lock Prompt。]

【卖点表达】
这张图只突出一个核心卖点：[核心卖点]。用户看完应理解：[用户要理解什么]。

【构图与镜头】
[镜头角度、主体位置、景别、留白、视角、焦点关系。]

【文字要求】
[如需文字，写明文案、位置、字体气质和排版方式。若未明确要求，写：不添加任何文字。]

【负面词】
[通用负面词 + 产品专属负面词。]
```

## Master Visual Direction

```text
【Master Visual Direction】
后续所有图片延续已确认主视觉的整体方向：[总结主视觉实际使用的背景、光影、色彩、构图、材质和画面气质]。

【允许变化的范围】
后续图片可以根据单张卖点调整[场景/局部构图/道具/文字位置/景别/使用情境]，但所有变化必须服务本张图的核心卖点。

【不允许变化的内容】
不得改变[背景色系/光影方向/材质呈现方式/产品主体地位/整体商业摄影质感/系列统一性]，不得让场景或道具抢过产品主体。
```
