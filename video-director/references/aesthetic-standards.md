# Aesthetic And Directing Standards

Use these standards for both Phase 1 concept judgment and Phase 2 prompt detail.

## Three Laws

### 1. Functional Scenarization

Always show function through story and scene. Do not directly state feature parameters.

Bad:

```text
支持实时翻译
```

Good:

```text
巴黎街头，老人微笑说法语，字幕自动浮现，用户露出惊讶表情。
```

If a function cannot be effectively dramatized, tell the user why and ask whether to revise the concept. Do not force a false scene.

### 2. Negative Space Is A Narrative Tool

Negative space is not emptiness; it is an active directing choice.

Allowed:

- Three seconds of silence
- Only wind sound
- No product appearance
- A single sentence
- Restrained non-key shots

The ratio between breath and density must be deliberate.

### 3. Camera Movement Requires Emotional Motivation

Every movement must have a reason.

Common emotional tendencies:

- Push in: discovery, approach, pressure
- Pull out: distance, release, awe
- Pan: exploration, observation
- Rise or descend: perspective shift, elevation
- Orbit: subject emphasis
- Locked shot: restraint, silence, weight

Before writing a camera move, answer: why push, pull, pan, move, rise, descend, orbit, or hold still?

## Image Aesthetic Standard

Each frame should carry international-brand advertising completion:

- Designed lighting: source quality, direction, intensity, and shadow behavior.
- Real spatial depth: foreground, subject, and background relationships.
- Cinematic composition: golden ratio, thirds, symmetry, frame-within-frame, leading lines, or another deliberate structure.
- Precise color control: palette, contrast, saturation, and color temperature.
- Clear visual memory point from emotion, character, scene, action, composition, light, or concept.

Do not force spectacle. Key shots should have poster-level memorability; non-key shots may be restrained and breathable.

## Lens Language Standard

Lens language must show:

- Strong rhythm across wide, medium, close, and detail emphasis.
- Deliberate negative-space density.
- Camera paths that are functional and beautiful.
- Fine acceleration and deceleration where movement is used.
- Industrial-level intentionality.
- Efficient visual storytelling and aesthetic control.

## Avoid Cliches

Do not add these for fake cinematic effect unless they truly serve the current narrative:

- Slow motion
- Cloud seas
- Starry skies
- Running
- Looking back
- Backlit silhouettes
- Water-drop close-ups
- City timelapse

Cinematic quality comes from narrative, emotion, and lens logic, not visual decoration.

## Boundaries

- If input is incomplete, ask for missing information.
- If a function cannot be dramatized, tell the user and ask for a decision.
- Do not output flat, bland, low-tension scripts.
- Do not directly recite product feature parameters.
- Do not contradict user-provided information.
- Do not skip Phase 1 unless explicitly requested.
- If the user dislikes Phase 1, create new directions instead of reusing the same idea.
- If the user changes product information midway, re-evaluate the creative direction.
- For videos of five seconds or less, focus on one emotional impact point; use minimalism and negative space.

## Final Self-Check

Check before output:

- Does each shot's movement have an emotional motivation?
- Is there negative-space design and breath?
- Are functions scenarized?
- Does the rhythm include prelude, climax, and aftertaste?
- Do key images have poster-level freeze-frame power?
- Are light, color, and texture executable?
- If narration exists, does it have poetic tension?
- Does the whole piece reach international-brand advertising completion?
